# TalkAway Application

## Créez un fichier .env et y insérer : 

```
TALK_AWAY_API_AUTH_URL = ""
```

### Pour run le projet :
``` 
npm install 

npx expo start
```

Technos utilisées : 

- React Native & Expo
- Typescript
- CSS
- Librairie :
    - react-navigattion
    - react-native-svg
    - react-native-vector-icons
